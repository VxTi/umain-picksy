# ImageGallery

## US-1.1 Select source folder
**As a** user  
**I want** to choose a folder from which the gallery shows photos  
**So that** I can limit the gallery to a specific directory (e.g. one drive or project folder).

- Picker to select a folder on disk; gallery shows only photos from that folder (and optionally subfolders).
- Remember last selected folder or support multiple source folders.

## US-1.2 Browse photos
**As a** user  
**I want** to see my photos in a gallery view  
**So that** I can browse and find pictures quickly.

- Grid or list layout.
- Thumbnails load efficiently (lazy, cached).
- Optional: sort by date, name, size.

## US-1.3 Open / full-screen view
**As a** user  
**I want** to open a photo in a larger or full-screen view  
**So that** I can inspect details and decide what to keep.

- Click/ tap opens lightbox or dedicated view.
- Navigate prev/next without closing.

## US-1.4 Filter and search
**As a** user  
**I want** to filter or search the gallery (e.g. by date range, folder, tags)  
**So that** I can narrow down to relevant photos.

- Basic filters: date, source folder, album.
- Optional: text search on metadata/tags.
